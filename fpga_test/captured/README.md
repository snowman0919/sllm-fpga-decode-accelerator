# Captured Board Outputs

Store raw DE10-Lite validation evidence here.

Expected generated file:

- `fpga_validation_summary.md` from `just fpga-report`

Recommended manual artifacts:

- board photos that show the programmed setup
- close-up photos where `HEX3..HEX0 = F F E A` is readable
- short videos of rerun or reset behavior
- logic-analyzer notes or handwritten observations

Placeholders:

- `de10_lite_validation_photo.placeholder.md` stays here until real board evidence is added

Use this directory for raw capture first. If one image becomes paper-ready, copy that selected asset into `paper_assets/figures/`.
