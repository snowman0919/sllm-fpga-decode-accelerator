# DE10-Lite Quartus Project Skeleton

This directory is a placeholder for the actual Quartus project used to validate the small decode-stage hardware block on the DE10-Lite.

Recommended contents after manual setup:

- `.qpf`
- `.qsf`
- synthesis reports
- programming files generated by Quartus

Use generated HDL from `../generated_verilog/` or `../../spinal/generated/`.
