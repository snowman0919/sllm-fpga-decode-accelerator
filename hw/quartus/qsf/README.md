# QSF Notes

Store verified or reference `.qsf` fragments here once you have board-checked pin assignments.

Do not guess final DE10-Lite pin assignments in this skeleton. Import them from:

- a verified board reference project, or
- a Quartus System Builder-generated design

At minimum, the final `.qsf` should cover:

- `CLOCK_50`
- `HEX0` to `HEX5`
- `LEDR`
- `SW`
- `KEY`
